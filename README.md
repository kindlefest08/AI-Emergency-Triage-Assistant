# AI-Based Emergency Triage Assistant

This project is designed to support healthcare workers in prioritizing patients during emergency situations using artificial intelligence. 

## Features
- **Triage Priority Prediction**: Machine learning to classify urgency (critical, urgent, non-urgent).
- **Symptom NLP Analyzer**: NLP-based processing of natural language symptoms.
- **Voice Symptom Input**: Speak your symptoms, converted to text via SpeechRecognition.
- **Explainable AI (XAI)**: Understand why a decision was made.
- **Queue Management**: Prioritized patient queues.
- **Doctor Recommendation**: Rules-based routing to specialists.
- **Patient Deterioration Prediction**: Predicting risk from vital signs.

## Setup
1. `pip install -r requirements.txt`
2. `python training/train_triage_model.py`
3. `python training/train_deterioration_model.py`
4. `python app.py` (or `streamlit run dashboard/streamlit_app.py`)
