# AI-Based Emergency Triage Assistant

An AI-powered system designed to assist in emergency medical triage by predicting patient deterioration and triage levels using machine learning models.

## Features
- Predicts patient deterioration risk
- Suggests triage priority level
- Uses machine learning models trained on healthcare data
- Modular architecture for preprocessing, feature engineering, and prediction
